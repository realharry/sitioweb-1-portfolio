# Portafolio Profesional - Sitio Web en Español para Ciudad de México

Un sitio web de portafolio profesional completamente desarrollado en español, dirigido a creativos y profesionales que viven en Ciudad de México. Construido con tecnologías modernas para artistas, músicos indie, diseñadores web, programadores, ingenieros de IA y profesionales de marketing.

## 🎨 Características Principales

- **Diseño Moderno y Colorido**: Tema vibrante con gradientes que reflejan la cultura mexicana
- **Completamente Responsivo**: Optimizado para dispositivos móviles, tablets y desktop
- **Modo Oscuro/Claro**: Interruptor de tema dinámico
- **Interfaz en Español**: Todo el contenido y la interfaz están en español para el mercado mexicano
- **Navegación Fluida**: Navegación suave entre secciones con enlaces de ancla

## 🚀 Tecnologías Utilizadas

- **Next.js 14**: Framework de React para producción
- **TypeScript**: Tipado estático para mejor desarrollo
- **Tailwind CSS**: Framework CSS utilitario para diseño responsivo
- **Shadcn UI**: Componentes de interfaz modernos y accesibles
- **Lucide React**: Iconos SVG optimizados
- **Next-themes**: Gestión del tema oscuro/claro

## 🛠️ Instalación y Uso

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/realharry/sitioweb-1-portfolio.git
   cd sitioweb-1-portfolio
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo**
   ```bash
   npm run dev
   ```

4. **Construir para producción**
   ```bash
   npm run build
   npm start
   ```
